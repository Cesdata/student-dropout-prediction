# Student Dropout & Academic Success Prediction

## 📋 Project Overview
This project aims to reduce academic dropout and failure in higher education. Using Machine Learning, I developed a classification model to identify students at risk at an early stage of their academic path. This allows educational institutions to implement support strategies before the student decides to leave.

## 📊 About the Dataset
The dataset includes 81 features known at the time of enrollment, covering:
* **Academic Path:** Previous qualifications and performance.
* **Demographics:** Age, gender, and nationality.
* **Socio-economic Factors:** Parents' occupation and educational background.

**Target:** A three-category classification task (**Dropout, Enrolled, and Graduate**).

## 🛠️ Tech Stack
* **Python** (Google Colab)
* **Scikit-Learn** (Classification Models)
* **Pandas & Numpy** (Feature Engineering)
* **Matplotlib/Seaborn** (Exploratory Data Analysis)

## 🚀 Key Objectives
* Identify the primary drivers of student attrition.
* Build a robust model to predict student outcomes with high accuracy.
* Provide actionable insights for academic retention strategies.
